# Personal Portfolio Website

## Overview

This project is a personal portfolio website developed using HTML and CSS. The website showcases my profile, skills, projects, and contact information in a professional and responsive design.

## Features

* Responsive web design
* Fixed navigation bar with smooth scrolling
* Professional hero section with profile image
* About Me section
* Skills section
* Projects section
* Contact form
* Hover effects and animations
* Modern UI with custom color scheme

## Technologies Used

* HTML5
* CSS3
* Google Fonts (Poppins)
